# Agent Instructions

This repository is an iterative performance piece. Each agent run receives the
current version of a working FizzBuzz project and is asked to make it more
enterprise-grade.

For each run:

1. Make one coherent round of changes.
2. Preserve the externally observable FizzBuzz behavior.
3. Keep or improve automated verification.
4. Build on previous iterations; do not simplify or reset their work.
5. Do not push to a remote or access anything outside this repository.
6. Stop after implementing and verifying the round.
